프로젝트 개요

반도체 제조를 위한 SECS/GEM 통신

POLAREX-200 CMP SECS/GEM 프로젝트는 SEMI E5 및 E30 표준을 구현하여 화학적 기계적 연마(CMP) 장비가 공장 호스트 시스템과 효과적으로 통신할 수 있게 합니다.

이 구현은 첨단 반도체 제조 시설에서 장비 제어, 모니터링, 데이터 수집 및 공정 관리를 위한 강력한 프레임워크를 제공합니다.

주요 기술:

  • SEMI E5 SECS-II 메시지 프로토콜
  • SEMI E30 GEM 표준
  • SEMI E37 고속 SECS 메시지 서비스 (HSMS)
  • SEMI E37.1 HSMS 단일 세션 모드
  • 장비 상태 추적을 위한 종합적인 상태 모델
  • 이벤트 기반 데이터 수집 시스템

SECS 메시지 구현

POLAREX-200 CMP SECS/GEM 인터페이스는 SEMI E5 표준에 따라 포괄적인 SECS-II 메시지 세트를 구현합니다. 이러한 메시지는 장비 상태 보고, 제어 기능, 알람 관리 및 공정 데이터 교환을 제공합니다.

카테고리 메시지 설명
장비 상태 S1F1/F2, S1F3/F4, S1F11/F12 기본 장비 상태 모니터링 및 온라인 확인
통신 제어 S1F13/F14, S1F15/F16, S1F17/F18 장비와 호스트 간의 통신 설정 및 제어
장비 제어 S2F41/F42, S2F13/F14, S2F15/F16 원격 장비 명령 전송 및 장비 상수 관리
데이터 수집 S2F33/F34, S2F35/F36, S2F37/F38 이벤트 기반 데이터 수집 구성 및 제어
알람 관리 S5F1/F2, S5F3/F4, S5F5/F6 장비 알람 보고 및 관리
공정 프로그램 S7F23/F24, S7F25/F26, S7F19/F20 호스트와 장비 간의 공정 레시피 관리 및 전송

상태 모델 구현

POLAREX-200 CMP SECS/GEM 인터페이스는 장비 상태와 동작에 대한 완벽한 가시성을 제공하는 5가지 종합적인 상태 모델을 구현합니다.

통신 상태 모델

장비와 호스트 간의 통신 상태를 추적합니다:

  • DISABLED: 통신 없음
  • ENABLED: 통신 존재
  • NOT COMMUNICATING: 연결은 설정되었으나 활성 통신 없음
  • COMMUNICATING: 완전한 통신 활성화

제어 상태 모델

호스트에서 사용할 수 있는 제어 수준을 정의합니다:

  • OFFLINE: 로컬 작업자만 장비 운영 가능
  • ONLINE/LOCAL: 호스트는 데이터를 얻을 수 있으나 제어는 불가
  • ONLINE/REMOTE: 호스트가 "전면 패널" 제어 권한 보유

공정 작업 상태 모델

처리 작업의 생명주기를 추적합니다:

  • QUEUED: 실행 대기 중인 작업
  • ACTIVE: 처리 중인, 활성 작업
  • EXECUTING: 진행 중인 공정 실행
  • PROCESS COMPLETE: 처리 완료됨

웨이퍼 공정 상태 모델

개별 웨이퍼의 상태를 추적합니다:

  • NEEDS PROCESSING: 처리 대기 중인 웨이퍼
  • IN PROCESS: 처리 중인 웨이퍼
  • PROCESSING COMPLETE: 웨이퍼 처리 완료
